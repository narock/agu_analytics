A heat map is a graphical representation of data where matrix values are represented as colors. Here, the matrix is the keyword usage by ESIP members over time. Each row contains a category of AGU keywords and each column contains a year. Colors represent the number of times a keyword from the category was used by an ESIP attendee to tag their AGU abstract. "Cooler" colors like greens and yellows indicate limited usage. "Hotter" colors like oranges and reds indicate extensive usage of that keyword category.
